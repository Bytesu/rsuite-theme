#RSuite Theme For Pagurian

这是一款基于[bootstrap](https://github.com/twbs/bootstrap)修改的,为[rsuite](http://rsuite.github.io)量身打造的[pagurian](https://github.com/hypers/pagurian)主题

## 生成样式
```bash
$ npm install
$ grunt
```

## 生成Demo站点
```bash
$ grunt exampleSite
```
